# MarkovGen
根据 [n-元语法](https://zh.wikipedia.org/zh-cn/%E8%AA%9E%E8%A8%80%E6%A8%A1%E5%9E%8B)（n = 2 或 3，再高会和训练集过于相似）模型随机生成文章。  
`train.txt` 是训练集（来源于雅思大作文考官范文），其中的换行符会被忽略。  
目前只支持英语。  
代码遵循 [Google Java Style](https://google.github.io/styleguide/javaguide.html)（借助 [google-java-format](https://github.com/google/google-java-format)）。

## 使用方法
点击 `Generate!` 即可。
